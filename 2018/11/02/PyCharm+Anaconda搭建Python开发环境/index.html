PyCharm+Anaconda搭建Python开发环境

2018-11-02 技术 PyCharm, Python Comments

下载与安装

Anaconda是一个开源的Python发行版本，PyCharm是一种常用的Python IDE。

下载Anaconda和PyCharm，选择合适版本安装。本文中使用默认安装路径。

PyCharm设置解释器

新建工程（本文工程名为“test”），该界面可在“Exsiting interpreter”处设置解释器，选择Anaconda安装路径下的解释器。或者在建立工程后，可在File->Settings->Project:test->Project Interpreter处设置。

测试程序

工程名处右键->New->Python File，输入程序。
